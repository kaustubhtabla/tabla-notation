Features

Everything You Need for Tabla Notations

Bhatkhande.io provides practical tools for creating, managing, and exporting Tabla compositions in a clean, focused Bhatkhande notation workflow.

Bhatkhande Notation Editor

Create and edit Tabla notation in a structured grid that matches the way compositions are actually taught and read.

Quick Bol Entry

Type bols rapidly, move through matras quickly, and keep your attention on the composition rather than formatting.

Composition Management

Save compositions with metadata like title, taal, type, laya, gharana, guru, and notes in one place.

Multiscript Support

View the same notation in English and Hindi without maintaining separate copies for different learners.

Export Options

Download compositions as PDF or JSON for printing, sharing, backup, and offline reference.

Teacher-Ready Workflow

Prepare classroom handouts, practice sheets, and personal repertoire archives with clean, readable output.

How It Works

Create Notations in Three Simple Steps

Start from the Hub, write bols in the editor, and export a clean PDF when you're ready to print or share.

Step 1

Start from the Hub

Choose the taal and composition type, then jump into the workspace with the structure set up for you.

Step 2

Write in the Editor

Enter bols using the grid and quick entry bar, then build sections and avartans as your composition grows.

Step 3

Export a Clean PDF

Generate a printable layout for practice, classroom handouts, or your personal archive.
